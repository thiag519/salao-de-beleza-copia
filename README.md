# Projeto - Salão de Beleza
Este é um projeto desenvolvido para um salão de beleza, com foco em apresentar serviços, equipe e formas de contato de maneira moderna, responsiva e agradável.
O site foi construído utilizando Next.js, React, TypeScript e Tailwind CSS.

#### ⚠️ Este repositório é uma cópia do projeto original, criado apenas para fins de demonstração e portfólio.

## Tecnologias utilizadas
**Next.js** – Framework React para aplicações modernas e otimizadas

**React** – Biblioteca para criação de interfaces dinâmicas

**TypeScript** – Tipagem estática para código mais seguro e escalável

**Tailwind CSS** – Estilização rápida e responsiva com utilitários CSS

## Funcionalidades principais
- Exibição dos serviços oferecidos pelo salão (cabelereiro, maquiagem, manicure etc.)

- Seção sobre o salão e apresentação da equipe

- Área de serviço com cada serviço oferecido por cada profissional

- Serviços oferecidos podendo ser selecionado e enviado com uma mensagem contendo valor, serviços e nome para o whatsApp

- Layout totalmente responsivo, adaptado para desktop e mobile

- Componentes reutilizáveis para fácil manutenção e expansão do site

## 🧩 Estrutura do projeto

```
├── public/           # Imagens e ícones
├── src/
│   ├── components/   # Componentes reutilizáveis
│   ├── area/         # Área de seções do site 
│   ├── context/      # Compartilhar dados entre componentes

│   ├── app/          # Páginas do Next.js
│   ├── styles/       # Estilos globais
│   ├── utils/        # Funções auxiliares
│   └── types/        # Tipos e interfaces TypeScript
└── package.json
```
## ⚙️ Como executar o projeto
#### 1. Clone este repositório:

Copiar código:

``git clone https://github.com/seu-usuario/salao-de-beleza-copia.git``
#### 2. Acesse a pasta do projeto:

Copiar código:

``cd salao-de-beleza-copia``
#### 3. Instale as dependências:

Copiar código:

``npm install``

 ou
 
``yarn install``

#### 4. Execute o servidor de desenvolvimento:

Copiar código:

``npm run dev``

 ou
 
``yarn dev``

#### 5. Abra no navegador:

Copiar código:

``http://localhost:3000``
