💇‍♀️ Projeto - Salão de Beleza
Este é um projeto desenvolvido para um salão de beleza, com foco em apresentar serviços, equipe e formas de contato de maneira moderna, responsiva e agradável.
O site foi construído utilizando Next.js, React, TypeScript e Tailwind CSS.

⚠️ Este repositório é uma cópia do projeto original, criado apenas para fins de demonstração e portfólio.

🚀 Tecnologias utilizadas
Next.js – Framework React para aplicações modernas e otimizadas

React – Biblioteca para criação de interfaces dinâmicas

TypeScript – Tipagem estática para código mais seguro e escalável

Tailwind CSS – Estilização rápida e responsiva com utilitários CSS

💡 Funcionalidades principais
Exibição dos serviços oferecidos pelo salão (cabelereiro, maquiagem, manicure etc.)

Seção sobre o salão e apresentação da equipe

Formulário de contato para agendamento ou dúvidas

Layout totalmente responsivo, adaptado para desktop e mobile

Componentes reutilizáveis para fácil manutenção e expansão do site
